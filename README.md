# MSSV_Assignment_DSA

Dự án thiết kế cấu trúc dữ liệu và giải thuật cho hệ thống hậu cần POLY-SHIP.

## Cấu trúc thư mục
- `routing.py`: Giải thuật Dijkstra và Kruskal.
- `hashing_tools.py`: Bảng băm và xử lý chuỗi (Rolling Hash).
- `promo_optimizer.py`: Quy hoạch động (Knapsack 0/1).
- `main_assignment.py`: File chạy chính.

## Cách chạy dự án
1. Mở terminal/command prompt tại thư mục chứa dự án.
2. Chạy lệnh: `python main_assignment.py`
3. Nhập các số từ 1 đến 8 để kiểm tra từng chức năng tương ứng trong menu.

## Nguồn tham khảo
- Thuật toán DSU & Dijkstra tham khảo từ bài giảng môn học (Slide bài 1-8).
